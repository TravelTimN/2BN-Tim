About

I am a Full-Stack Web Developer, with a passion for Front-End UI/UX Design. I love to build pixel-perfect responsive websites and applications, using a combination of semantic code with modern technologies. In addition to web development, I enjoy graphic design, with an emphasis on Adobe software. Whether it's building a wireframe, creating a simple blog, designing a social media avatar, or coding a full-stack website, I'm available for freelance work, so please contact me!
